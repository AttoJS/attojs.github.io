# 默认请求

我们内置了 Fetch 来处理你的请求，你可以借助第一个参数传入一个 字符串、一个对象或者传入一个请求函数

# 传入字符串

你可以通过传入一个接口的链接

import { useRequest } from 'vue-request';

const { data } = useRequest('api/user?name=John');

# 传入请求函数【推荐】

可以通过传入一个返回值为 Promise 类型的函数。

换句话说，你可以使用 第三方的请求库（如 axios ） 来获取数据，然后将该请求传给 VueRequest 来进行管理。 具体可参考以下例子

import { useRequest } from 'vue-request';
import axios from 'axios'

const getUser = () => {
  return axios.get('api/user');
};

const { data } = useRequest(getUser);

# 传入对象

你可以通过传入一个包含 url 属性的对象

import { useRequest } from 'vue-request';

const objectService = {
  url: 'api/user',
  method: 'POST',
  body: JSON.stringify({
    name: 'Benny',
  }),
  headers: new Headers({
    'Content-Type': 'application/json',
  }),
};

const { data } = useRequest(objectService);
