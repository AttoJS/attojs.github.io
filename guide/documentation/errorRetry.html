# Error Retry

In applications, transient errors are actually very common. For example, the connection between the interface server and the database server is temporarily disconnected, or the user's network temporarily fails. These failures are often corrected by themselves in a short time. If you can try again after a suitable delay, the request may be successful.

Now you can let VueRequest handle this for you through a simple configuration. Just provide a errorRetryCount to tell us the number of retries. As the following example

# ErrorRetryCount

import { defineComponent, ref, watchEffect } from 'vue';
import { useRequest } from 'vue-request';
import Mock from 'mockjs';

let error = true;

function testService() {
  return new Promise((resolve, reject) => {
    if (error) {
      setTimeout(() => {
        reject('Request failed');
      }, 2000);
    } else {
      setTimeout(() => {
        resolve(Mock.mock('@name'));
      }, 1000);
    }
  });
}

export default defineComponent({
  name: 'App',
  setup() {
    const errorCount = ref(0);
    const isErrorRequest = ref(error);
    const { data, loading, run } = useRequest(testService, {
      errorRetryCount: 5, // it will retry 5 times
      onError: () => errorCount.value++,
      onSuccess: () => (errorCount.value = 0),
    });

    watchEffect(() => {
      error = isErrorRequest.value;
    });

    return () => (
      <div>
        <button onClick={() => (isErrorRequest.value = !isErrorRequest.value)}>
          switch to <b>{isErrorRequest.value ? 'success' : 'error'}</b> request
        </button>
        &nbsp;&nbsp;&nbsp;
        <button
          onClick={() => {
            errorCount.value = 0;
            run();
          }}
        >
          run
        </button>
        <br />
        <br />
        <div style="height: 32px">{errorCount.value > 0 && `error times: ${errorCount.value}`}</div>
        <br />
        <div style="height: 32px">
          {loading.value ? (errorCount.value ? 'retry loading...' : 'loading...') : data.value}
        </div>
      </div>
    );
  },
});


























 

































# ErrorRetryInterval

Tips

By default, we will use Exponential Backoff Algorithmopen in new window to help you calculate the appropriate interval time

Of course, you can also provide errorRetryInterval to set the retry interval

import { defineComponent } from 'vue';
import { useRequest } from 'vue-request';

export default defineComponent({
  name: 'App',
  setup() {
    const { data } = useRequest('api/users', {
      errorRetryCount: 5, // it will retry 5 times
      errorRetryInterval: 3000, // The retry interval is 5 seconds
    });
    return () => <div>{data.value}</div>;
  },
});
