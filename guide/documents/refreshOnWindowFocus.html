# 屏幕聚焦重新请求

有些时候，你要确保多个浏览器窗口之间数据的一致性；又或者是当用户电脑在休眠状态重新激活后，页面的数据需要同步到最新状态时。refreshOnWindowFocus 可能会为你节省很多逻辑代码，如下所示，你只需要启用 refreshOnWindowFocus 即可








 





import { defineComponent } from 'vue';
import { useRequest } from 'vue-request';

export default defineComponent({
  name: 'App',
  setup() {
    const { data } = useRequest(() => `api/users`, {
      refreshOnWindowFocus: true,
    });
    return () => <div>{data.value}</div>;
  },
});

# refocusTimespan

如果你不希望用户在短时间内频繁触发 refreshOnWindowFocus 导致频繁请求的话，你可以修改 refocusTimespan 的值，从而延长触发的间隔，如下所示








 
 





import { defineComponent } from 'vue';
import { useRequest } from 'vue-request';

export default defineComponent({
  name: 'App',
  setup() {
    const { data } = useRequest(() => `api/users`, {
      refreshOnWindowFocus: true,
      refocusTimespan: 5000,
    });
    return () => <div>{data.value}</div>;
  },
});
