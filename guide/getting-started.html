# Getting Started

前提条件

确保 Vue 版本为 3.x

# Installation

npm install vue-request
# Or with yarn:
yarn add vue-request

# CDN

提醒

请确保在引入 VueRequest 前先引入 Vue

如果你想在页面上以 CDN package 的形式导入，我们提供了对应的 CDN 版本


 


<!-- Please make sure to import Vue before introducing VueRequest -->
<script src="https://unpkg.com/vue@next"></script>
<script src="https://unpkg.com/vue-request.min.js"></script>

# Example
















 










import { defineComponent } from 'vue';
import { useRequest } from 'vue-request';

// Mock a request
function testService() {
  return new Promise<string>(resolve => {
    setTimeout(() => {
      resolve('success');
    }, 1000);
  });
}

export default defineComponent({
  name: 'App',
  setup() {
    const { run, data, loading } = useRequest(testService);
    return () => (
      <div>
        <button onClick={() => run()}>run</button>
        <br />
        {loading.value ? 'loading...' : data.value}
      </div>
    );
  },
});
